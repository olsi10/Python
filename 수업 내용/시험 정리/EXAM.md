# 파이썬

파이썬 언어는 오픈 소스 프로젝트로서 무료 배포된다.
최근 파이썬을 이용해 프로그램을 개발하는 기업이 많아졌다.

<br />
<br />

# 파이썬의 종류

| 종류       | 기능                                                            |
| :--------- | :-------------------------------------------------------------- |
| CPython    | C언어로 작성한 파이썬 기본적으로 파이썬은 CPython               |
| Jython     | Java로 작성된 파이썬 JVM에서 동작                               |
| IronPython | .NET 프레임워크와 Mono용으로 개발한 파이썬, C#으로 작성         |
| PyPy       | Python으로 작성한 파이썬, CPython보다 빠르게 수행되는 것을 목표 |

# 파이썬 장점

| 장점                          | 내용                                                                                            |
| :---------------------------- | :---------------------------------------------------------------------------------------------- |
| 오픈 소스                     | 비용 X, 전세계 개발자가 참여하기 때문에 점점 발전                                               |
| 쉽고 간결한 문법              | 다른 언어들과 달리 문법이 간결하기 때문에 쉽게 배울 수 있다. 기존 언어 개념만 안다면 쉽게 습득! |
| 풍부한 라이브러리             | 기본적으로 제공되는 라이브러리가 풍부, 이미 만들어진 라이브러리를 쉽게 추가하여 구현 가능       |
| C와 쉬운 연계                 | C언어로 구현된 부분과 잘 연결되므로 풍부하고 다양한 개발 가능, 속도 Up                          |

| 유니코드 사용                 | 모든 언어를 자유롭게 사용가능                                                                   |
| 빠른 개발 속도                | 간결한 문법과 많은 라이브러리를 통해 빠르게 개발 가능                                           |
| 대부분 운영체제에서 실행 가능 | 대부분 운영체제에서 같은 방법으로 실행 가능                                                     |

## 정수를 표현하는 여러 가지 방법

"{:d}".format(515)

"{5:d}}".format(515) # 최소 5칸 차지 후 정수 넣기

"{:+5d}".format(515) # 양수면 + 표시

"{:=+5d}".format(515) # + 맨 앞에

"{:05d}".format(515) # 빈칸은 0

"{:+05d}".format(515) # 양수면 0 앞에 + 표시

## 실수를 표현하는 여러 가지 방법

"{:f}".format(11.12)

"{:12f}".format(11.12) # 최소 12칸 차지 후 실수 넣기

"{:12.1f}".format(11.12) # 소수점 첫째자리까지 반올림

# 셋 총 정리

<br />

- 중복없는 자료형이다. 수학에서와 같이 중괄호 사용
- 집합과 같은 개념
- 순서, 중복 X, 다양한 연산

```python
game = {"LOL", "PIPA", "RIGDER", 1239}

set("Funny") # 'n', 'F', 'y', 'u'
```

- set(문자열) = 문자열의 문자가 하나씩 셋의 요소로 들어간다.
- set(리스트) = 리스트의 각 요소가 하나씩 셋으로 들어간다.
- set(튜플) = 튜풀의 각 요소가 하나씩 셋으로 들어간다
- set(딕셔너리) = 딕셔너리의 키가 각 요소로 하나씩 들어간다.
  -set(range()) = range() 함수의 각 요소가 셋에 하나씩 들어간다.

# 프로그래밍 패러다임

<br />

| 종류                 | 내용                                                                                                                                                                                                                                                                                                      |
| :------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 순차적 프로그래밍    | 컴퓨터가 할 일을 명령어 순서대로 나열, 그 명령어를 차례대로 실행되는 프로그램을 개발하는 법 <br />조건에 따라 다르게 실행할 수 있는 조건문, 반복문 사용                                                                                                                                                   |
| 절차적 프로그래밍    | 프로시저, 즉 함수르 기본으로 프로그래밍 개발, 함수에 실행할 절차를 명령어로 작성<br />데이터를 함수에 넣어 실행할 결과를 처리하는 프로그램을 작성하는 방법                                                                                                                                                |
| 구조적 프로그래밍    | 절차와 비슷하지만 단위가 모듈이다. 모듈은 보통 파일이며 관련 있는 함수들을 모아 놓은 것이다.<br />이 파일을 가져다 사용하면 재사용성을 높인 프로그래밍 방법이다                                                                                                                                           |
| 객체 지향 프로그래밍 | 절차, 구조 프로그래밍은 기본이 함수나 모듈이며 명령어를 모아 놓고 데이터를 처리하는 것이 바탕이다.<br />하지만 이런 함수나 모듈도 같이 다뤄야 할 밀접한 관계가 있는 데이터가 존재.<br />객체 지향 프로그래밍은 이렇게 관련 있는 데이터와 함수를 모아 객체로 만들고 그 객체를 기반으로 프로그래밍하는 방법 |
